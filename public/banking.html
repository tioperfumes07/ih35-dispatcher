Bank transactions & QuickBooks matching

Pull Purchases, Bills, and Bill Payments from QuickBooks for a date window; paste bank CSV (Date, Amount, Description); run suggested matches by amount and date. Saved links = this app’s audit trail only — not posted to QuickBooks. Links here tie a bank line to a QBO document for traceability — not QBO journal lines or bank-feed posts. Authoritative bank clearing stays in QuickBooks reconciliation.

1 · Refresh QBO + ERP snapshot
Snapshot summary line + metric tiles — filled only after Load snapshot. #btnSnap handler: #snapMeta textContent (window + ERP candidate count); #snapMetrics innerHTML of .metric cards from GET /api/banking/snapshot QBO counts + import/link tallies (escaped via esc()).
2 · Import bank CSV

Expected columns: Date, Amount (negative for outflows), and Description (or Memo). Tab or comma separated.

After Import rows, copy the import batch id from the result into step 3. Prefer YYYY-MM-DD for reliable parsing.
Import result HTML (row count + batch id) — empty until Import rows succeeds. #btnImport → POST /api/banking/import-csv; #importOut innerHTML with counts + <code> batch id; #batchId auto-filled for step 3.
3 · Suggest matches & link
Suggestion cards + pager — innerHTML from Run suggestions or pager slice. #btnSuggest → POST /api/banking/suggest; #suggestOut innerHTML (empty state or renderBankSuggestView() blocks); #suggestPagerHost via erpPagerRender. Uses esc() for cell text.
