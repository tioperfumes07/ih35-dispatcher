The loads board Auto miles feature uses road routing (OSRM) for practical miles and a straight-line approximation for shortest. For settlement-grade practical/short pairs, plan to connect Trimble PCMiler / Maps (commercial API). Set PCMILER_API_KEY or vendor-specific env vars when you license it — the same button can call a different calculator behind the scenes.
